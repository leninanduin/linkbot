## What it does?

replaces http://twitter.com/ACTION for tweetbot:///ACTION urls, ONLY FOR MAC.

## Works on:

- @username links & widget
- Tweet widget (with current paget title & url)
- Tweet to @USERNAME widget
- Tweet #HASHTAG widget
- Url to #HASHTAG
- YouTube sharing
- Redability sharing
- Pocket sharing
